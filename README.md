# AuroraOS_QML_components_prompt

Тут располагается текстовый промпт на английском для генерации QML компонент из изображений при помощи нейросетей Kimi и GLM.
Смотри промпт ниже
XXXX -заменить на общепринятое название элемента который приложен в изображении. 

======================== ПРОМПТ ==========================

Write QML component as visualized on a attached picture as a XXXX 
for Qt 5.6 for Aurora OS/Sailfish OS  using import QtQuick 2.0 and import Sailfish.Silica 1.0 and import QtQuick.Layouts 1.0 
write QML component as a library. 
Use an adaptive layout using anchors and layouts 
Using a Theme to select the size of elements and fonts and colors. 
Use lowercase letters for property names.

======================== ======= ==========================

Порядок использования 
1 ) приложить изображение графического элемента как картиину в промпт в web версии Kimi или GLM.
2 ) скопировать промпт выше и встравить в тектовое поле ввода пропта в web браузуре.
3 ) заменить - XXXX - на общепринятое название элемента который приложен в изображении.
4 ) нажать кнопку герерации 
5 ) если элемент не похож на исходник, попросить нейросеть "XXXX(элемент) не похож на оригинал , доработай"
